Second Life

Web App based on Google Maps API focused on giving objects no longer wanted by their owners a new home. As a user you will see a list of items and their locations, so you can easily go and rescue those objects and give them a Second Life.


We also provide the possibility to change into a map view, which will display an interactive map with a marker for every posted object, making it easier to search by area.

Hackathon Game

Project built for #hackforheroes, a 2.5 days hackathon organized by Makers. The main idea was creating a game to make the lockdown time pass a bit faster while tributing our key workers.


The first day we agreed the global mantra for this pandemic is 'We are all in this together' so making the game inclusive was our goal.


We can select the skin tone and gender of our doctor and will help him jump over the virus while changing continent on every level.

Mix Tape

Makers final project, built in 10 days with no prior experience on React or APIs.


Uses Spotify to authenticate users and lets you select your friends from a list to create a combined playlist with your group's top tracks from Spotify.


The playlist gets saved in Spotify and in our database, so it can be played directly.
